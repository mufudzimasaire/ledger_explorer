# XRP Ledger Explorer

This is a simple ReactJS based explorer for the XRP ledger. 

## Getting started

Install dependencies it and boot app:

```bash
-: yarn
-: yarn dev

# OR

-: npm install
-: npm run dev
```

## Example

The application only works with valid xrp ledger addresses, eg.

```bash
r9cZA1mLK5R5Am25ArfXFmqgNwjZgnfk59
```
